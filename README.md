# Laravel Subby

**Laravel Subby** is a flexible plans and subscription management system for Laravel. You can also schedule plan changes.

Originally forked from [rinvex/laravel-subscriptions](https://github.com/rinvex/laravel-subscriptions).

## Why this fork

This fork exists because the original package [bpuig/laravel-subby](https://github.com/bpuig/laravel-subby) is no longer maintained. It includes the following patches on top of upstream v7:

- **Laravel 11/12/13 support**: Updated PHP requirement to ^8.2 and relaxed framework constraints.
- **Type fixes**: Added missing property types in `IsPaymentMethod` trait, fixed `getKey` error.
